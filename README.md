# KyaniteShift GitHub Pages Landing Page

Static landing page for GitHub Pages.

## Publish
1. Create a new public GitHub repository, for example `kyaniteshift`.
2. Upload `index.html` to the repository root.
3. Go to Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Save. GitHub will provide a URL like:
   https://YOUR-USERNAME.github.io/kyaniteshift/

No API calls, backend, external images, or external assets.
